# Screenshots

Drop three PNGs here (exact filenames — the main README already references them):

| Filename | Capture |
|---|---|
| `markets.png` | `/markets` — the grid view, ideally with a couple of markets showing both crowd and AI odds |
| `market-detail.png` | `/markets/[slug]` — a single market with the AI rationale panel expanded |
| `leaderboard.png` | `/leaderboard` — showing the 🤖 AI Analyst row alongside human forecasters |

Quickest way to capture them:

```bash
npm run dev
```

Then visit each route in a browser at a reasonable width (~1280px) and screenshot the
page content. No resizing/cropping needed — GitHub scales images to fit the README table.
